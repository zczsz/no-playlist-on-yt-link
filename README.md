
sometimes large playlists on youtube slow down my computer, and autoplay is annoying as i might want to read 
the comments before jumping to the next video, so i made this extension.

to test  it go to
> about:debugging -> this firefox -> load temporary add-on.

right click the link to any video and click on "open without playlist", and it'll open the video in a new tab 
without the playlist (and actually it'll remove anything after the first & in the url).
